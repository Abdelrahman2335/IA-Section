# IA-Section

IA-Section is a simple PHP + MySQL web application for course registration.
Users can create an account, log in, and manage their registered courses from a protected home page.

## Features

- User authentication (register, login, logout)
- Protected home page (only accessible when logged in)
- Add a course from a dropdown list
- Edit a registered course using the same dropdown modal
- Delete a registered course with a warning confirmation
- Flash messages for success/error states (including duplicate course prevention)

## Tech Stack

- PHP
- MySQL / MariaDB
- Bootstrap 5
